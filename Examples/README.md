# Examples

| Project | Topic |
|---|---|
| `Asp.Net` | ASP.NET Core MVC integration with a Binance data provider |
| `Minimal WebApi` | Minimal ASP.NET Core API integration, dependency injection, and public price endpoint |
| `ConsoleClient` | Basic REST requests and WebSocket trade subscription |
| `Binance.Examples.OrderPlacement` | Spot and USD-M futures order placement, order status lookup, and cancellation |
| `Binance.Examples.OrderBook` | Client-side order book implementation with live console display |
| `Binance.Examples.Tracker` | Trade tracker with rolling market statistics |
| `Blazor.ClientSide`, `Blazor.ServerSide`, `Blazor.DataProvider` | Blazor examples and shared Binance data provider |
| `WpfClient` | WPF client example |
| `Serverless.BlazorWASM`, `Serverless.FunctionApp` | Serverless Blazor WebAssembly and Azure Functions examples |

The `ai-friendly` folder contains standalone copy/paste examples intended for quick onboarding and AI coding assistant context.
